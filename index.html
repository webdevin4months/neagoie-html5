When we submit form the info we entered gets sent to the server in a query string. As we are only practising our string is visible in the address bar. We used method GET by default which dumps the info in the address bar. the POST method wouldn't put the info in the address bar.

form action.php used with php sites.

?firstname=fake&lastname=man&email=fake%40man.com&password=123&gender=on&cat=on

url encoding replaces certain characters with a %.

what gets sent is in pairs name and value.

  1. firstname=fake
  2. lastname=man
  3. email=fake%40man.com
  4. password=123
  5. gender=on
  6. cat=on
