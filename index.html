Section 4. Course Notes

there's about 10-15 tags we'll use daily, just know where to look for the rest.

We use tags so the browser knows what to do with our content.

there is tag nesting. All the content you can see on the page is nested inside the body tags. the body is the parent tag.

When tags are used they become elements.

Strong tags bold/emphasise text. don't use bold tags.

em tags Italicize text.
